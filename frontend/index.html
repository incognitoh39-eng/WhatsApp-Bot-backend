01 · Enlazar worker

Genera tu worker de Termux

Escribe el número que usarás en WhatsApp (con código de país) y te damos el comando exacto para pegar en Termux.

Comando para pegar en Termux

  1. Instala Node.js en Termux si no lo tienes: pkg install nodejs git -y
  2. Pega y ejecuta el comando de arriba dentro de la carpeta del bot.
  3. Cuando la terminal muestre el código de vinculación, aparecerá también aquí abajo.
Esperando que ejecutes el comando en Termux…

Código para vincular en WhatsApp

----

WhatsApp → Dispositivos vinculados → Vincular con número de teléfono

02 · Motor de IA

Conecta tu API key de Groq

Tu WhatsApp ya está vinculado. Esta clave se envía directo a tu propio worker de Termux, nunca queda guardada en este servidor.

¿No tienes una? Consíguela gratis en console.groq.com/keys

03 · Control

Tus contactos

Activa la IA solo en las conversaciones donde quieras respuesta automática. Los grupos nunca reciben auto-respuesta.

Aún no llegan contactos desde tu worker. Escríbete algún chat en WhatsApp o pulsa «Refrescar».
